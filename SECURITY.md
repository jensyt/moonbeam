# Security Policy

## Supported Versions

Currently, Moonbeam is in early development. Only the latest version is supported with security updates.

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Moonbeam, please report it by emailing jensyt@gmail.com.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue (if possible).
- Any potential impact.

We will acknowledge your report and work towards a fix in a timely manner. We request that you do not disclose the vulnerability publicly until a fix has been released.
